# Scopo
I seguenti obiettivi anche se numerati possono essere realizzati in qualsiasi ordine.
**NOTA** Si ricorda di effettuare un buon numero di commit come da best practice di git.

## Obiettivo 1 - Implementazione Libreria
Creare un implementazione delle funzioni definite nel file libreria.h all'interno del file libreria.cpp

## Obiettivo 2 - Test della Libreria
Apportare le necessarie modifiche al file main.cpp per utilizzare la libreria appena implementata, ed eventualmente 
aggiungere nuovi istruzioni al main.cpp. Lo scopo è utilizzare il main.cpp per testare le funzioni contenute nella 
libreria

## Obiettivo 3 - Autori
I tutti i file che modificate modificare deve essere inserito all'inizio del file come commento la seguente riga:
```
// Autore: [Cognome] [Nome]
// Data:   2026/05/23
// Classe: 3CI
// Fila:   [Fila]
```
Sostituendo:
- `[Cognome]` con il proprio cognome.
- `[Nome]` con il proprio nome.
- `[Fila]` con la lettera della fila in cui che state svolgendo.
